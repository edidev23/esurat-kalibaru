Form Fields

Form fields should be wrapped in the following code:

<p class="form_line_field">
	<label class="form_line_left_field" for="FIELD_ID">FIELD_LABEL</label>
	<span class="form_line_right_field">
		FIELD_INPUT_TAG<br>
		<span class="helper_text">HELP_TEXT</span>
	</span>
	<br class="clear_both">
</p>


<input type="text" name="" value="" class="input_fld input_text" id="">


<input type="text" name="" value="" class="input_fld input_text_small" id="">


<input type="file" name="" class="input_file" id="">


<input type="text" disabled="disabled" name="" value="" class="input_fld input_text" id="">


<textarea rows="10" cols="20" name="" class="input_fld input_textarea" id="">


<textarea rows="10" cols="20" name="" class="input_fld input_textarea_large" id="">


<select name="" class="input_fld input_select" id="">


<select multiple="multiple" name="" class="input_fld input_select" id="">

Form Content

Form content should be wrapped in the following code:

<p class="form_line_content">
	<span class="form_line_left_content">CONTENT_LABEL</span>
	<span class="form_line_right_content">
		CONTENT<br>
		<span class="helper_text">HELP_TEXT</span>
	</span>
	<br class="clear_both">
</p>

Example 1: Ut at massa aliquam libero porttitor vehicula

Example 2: Suspendisse potenti

Tables

Table widths should generally be decided by the browser but you can add the CSS class width_full to your table if you'd like it to fill its parent.

Sub-header 1 Sub-header 2
Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2
