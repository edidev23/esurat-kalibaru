Same Height: Magic JavaScript

Quick Jump ↵

Resize a parent's kids to be the same height

Usage

Add data-mjf="same_height" id="ELEMENT_ID" to your parent tag.

Add data-mjf="same_height_child" to the children of your parent tag that you would like to be resized.

Add <script type="text/javascript">$m.wand.same_height = true;</script> inside your <body> tag.
