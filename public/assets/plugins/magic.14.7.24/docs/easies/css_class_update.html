CSS Class Update: Magic JavaScript

Quick Jump ↵

Easily change an element's CSS class

Introduction

First add data-mjf="css_class_update|return_false" data-mjf_ccu_id="ELEMENT_CONTENT_ID" id="ELEMENT_ID" to your clickable tag.

Next there are three types of changes you can invoke…

  1. add — data-mjf_ccu_exec="add(YOUR_CLASS_NAME)"
  2. replace — data-mjf_ccu_exec="replace(YOUR_CLASS_NAME)"
  3. remove — data-mjf_ccu_exec="remove(YOUR_CLASS_NAME)"

Finally add <script type="text/javascript">$m.wand.css_class_update = true;</script> inside your <body> tag.
