Text Resizer: Magic JavaScript

Quick Jump ↵

Usage

Add <script type="text/javascript">$m.wand.text_resize = XX;</script> inside your <body> tag where XX is the default size in pixels of the text.

Use the example below as a guide for building your own resizer!
