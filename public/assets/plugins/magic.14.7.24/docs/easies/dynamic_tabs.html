Dynamic Tabs: Magic JavaScript

Quick Jump ↵

Usage

Add <link href="/PATH/TO/magic/stylesheets/dynamic_tabs..css" rel="stylesheet">

below your <script type="text/javascript" src="/PATH/TO/magic/wand..js" charset="UTF-8"></script>

Add…

<script type="text/javascript">$m.wand.dynamic_tabs = true;</script>

-OR-

<script type="text/javascript">
 $m.wand.dynamic_tabs = [{
   name : "tab_ex"
  ,autostart : true
  ,delay : 3 // seconds
 }];
</script>

…inside your <body> tag.

Use the example below as a guide for building your own!

Notes

Add data-mjf_dtab_shct="KEYSTROKE" to your <a> tags if you'd like to enable keyboard shortcuts.

Add data-mjf_ga_track="false" to your <a> tags if you don't want Google Analytics to track the particular click.

Add data-mjf_dtab_cookie="false" to your <a> tags if you don't want the last tab to be remembered.
